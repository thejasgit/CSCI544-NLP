ಬೆಂಗಳೂರು: ಹಾಲು, ಅಡುಗೆ ಅನಿಲ ಬೆಲೆ ಏರಿಕೆ ಬಳಿಕ ಈಗ ವಿದ್ಯುತ್‌ ದರ ಏರಿಕೆಯ ಶಾಕ್‌! &nbsp;ಕರ್ನಾಟಕ ವಿದ್ಯುತ್ಛಕ್ತಿ ನಿಯಂತ್ರಣ ಆಯೋಗ (ಕೆಇಆರ್‌ಸಿ) ಮಂಗಳವಾರ 2017-18ನೇ ಸಾಲಿನ ಪರಿಷ್ಕೃತ ವಿದ್ಯುತ್‌ ದರ ಪ್ರಕಟಿಸಲಿದೆ.ಪ್ರತಿ ಯೂನಿಟ್‌ಗೆ 70 ಪೈಸೆಯಿಂದ 1 ರೂಪಾಯಿ ತನಕ ದರ ಹೆಚ್ಚಳವಾಗುವ ಸಾಧ್ಯತೆಯಿದೆ ಎಂದು ಸರಕಾರದ ಮೂಲಗಳು ಹೇಳುತ್ತಿವೆ.ವಿದ್ಯುತ್‌ ಬೇಡಿಕೆ ಹೆಚ್ಚಳ ಮತ್ತು ಅದರ ಪೂರೈಕೆಗೆ ಖರೀದಿ ಹಾಗೂ ಬರುವ ಆದಾಯದಲ್ಲಿ ಅಂತರ ಇದೆ.ಆದ್ದರಿಂದ ಪ್ರತಿ ಯೂನಿಟ್‌ಗೆ ಸರಾಸರಿ 1.40 ರೂ.
