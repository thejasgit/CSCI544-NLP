ರಾಯಚೂರು: ಕಿಡ್ನಿ ಸಮಸ್ಯೆಯಿಂದ ಬಳಲುತ್ತಿರುವ ಮಾನ್ವಿ ತಾಲೂಕು ಬೆಟ್ಟದೂರು ತಾಂಡಾ ನಿವಾಸಿಗಳ ಆರೋಗ್ಯ ವಿಚಾರಣೆಗೆ ಆರೋಗ್ಯ ಇಲಾಖೆ ದೌಡಾಯಿಸಿದೆ.ವರದಿಯಿಂದ ಎಚ್ಚೆತ್ತ ಇಲಾಖೆ, ತಾಂಡಾಕ್ಕೆ ಭೇಟಿ&nbsp;ನೀಡಿ ಜನರ ಆರೋಗ್ಯ ಪರೀಕ್ಷಿಸಿದೆ.ಮೂರು ತಂಡಗಳಿಂದ ತಾಂಡಾದ 300ಕ್ಕೂ ಅಧಿಕ ನಿವಾಸಿಗಳ ಆರೋಗ್ಯ ಪರೀಕ್ಷಿಸಲಾಗಿದೆ.ಇದು ಮೇಲ್ನೋಟಕ್ಕೆ ನೀರಿನಿಂದಾದ ಸಮಸ್ಯೆ ಎಂಬುದು&nbsp;ಖಚಿತವಾಗುತ್ತಿದೆ.ಬೆಟ್ಟದೂರು ತಾಂಡಾದ ಸಮಸ್ಯೆ ಹಿನ್ನೆಲೆಯಲ್ಲಿ ಗ್ರಾಮಕ್ಕೆ ತೆರಳಿ ನಿವಾಸಿಗಳ ತಪಾಸಣೆ ನಡೆಸಲಾಗಿದೆ.
