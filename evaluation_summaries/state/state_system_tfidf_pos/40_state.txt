ಬೆಂಗಳೂರು: ಹಾಲು, ಅಡುಗೆ ಅನಿಲ ಬೆಲೆ ಏರಿಕೆ ಬಳಿಕ ಈಗ ವಿದ್ಯುತ್‌ ದರ ಏರಿಕೆಯ ಶಾಕ್‌!  ಕರ್ನಾಟಕ ವಿದ್ಯುತ್ಛಕ್ತಿ ನಿಯಂತ್ರಣ ಆಯೋಗ (ಕೆಇಆರ್‌ಸಿ) ಮಂಗಳವಾರ 2017-18ನೇ ಸಾಲಿನ ಪರಿಷ್ಕೃತ ವಿದ್ಯುತ್‌ ದರ ಪ್ರಕಟಿಸಲಿದೆ. 
ಪ್ರತಿ ಯೂನಿಟ್‌ಗೆ 70 ಪೈಸೆಯಿಂದ 1 ರೂಪಾಯಿ ತನಕ ದರ ಹೆಚ್ಚಳವಾಗುವ ಸಾಧ್ಯತೆಯಿದೆ ಎಂದು ಸರಕಾರದ ಮೂಲಗಳು ಹೇಳುತ್ತಿವೆ.. 
ಮಂಗಳವಾರ ವಿದ್ಯುತ್‌ ದರ ಪರಿಷ್ಕರಣೆ ಮಾಡಿ ಆದೇಶ ಹೊರಡಿಸುವ ಸಾಧ್ಯತೆ ಇದೆ.. 
ಸಾಮಾನ್ಯವಾಗಿ ಮಾರ್ಚ್‌ 31ರ ಒಳಗೆ ವಿದ್ಯುತ್‌ ದರ ಪರಿಷ್ಕರಣೆ ಆದೇಶ ಹೊರಡಿಸಲಾಗುತ್ತದೆ.
